# ImageBox 版本变更记录

所有重要变更都会记录在此文件中。遵循 [Keep a Changelog](https://keepachangelog.com/) 格式。

## [0.0.2] - 开发中

### 新增
- **国际化支持 (vue-i18n)**
  - 添加英文 (en) 和中文 (zh-CN) 语言包
  - 应用内所有界面文本支持中英文切换
  - 默认语言为英文
- **设置面板 (Settings)**
  - 语言切换功能 (English / 中文)
  - 主题切换 (浅色 / 深色)
  - 版本信息展示
  - 设置持久化到 localStorage
- **帮助面板 (Help)**
  - 版本信息
  - 快速开始指南
  - Zoom 缩放使用说明
  - 快捷键说明
  - 支持的图片格式列表
  - 使用技巧

---

## [0.0.1] - 2024-03-27

### 已完成
- 项目初始化，Vue 3 + Vite
- Naive UI 组件库集成
- 基础 UI 布局 (顶部导航 + 侧边栏)
- 图片压缩功能实现
  - Canvas API 压缩
  - 质量调节
  - 多图批量处理
  - 支持 JPEG/PNG/WebP 输出
- 格式转换功能
- HEIC 格式支持 (heic-decode)
- 剪贴板图片粘贴
- Zoom 缩放支持

### 技术栈
- Vue 3.5 + Composition API
- Vite 5
- Naive UI 2.x
- Pinia 状态管理
- Vue Router 4
- @vicons/carbon 图标库
