# ImageBox 开发计划

## 项目概述

- **项目名称**: ImageBox (cb-imagebox)
- **当前版本**: 0.0.2
- **目标**: 一款轻量级图片处理工具箱，支持压缩、格式转换、水印、裁剪、截图美化等功能

---

## 功能规划

### 核心功能

| 功能 | 状态 | 优先级 | 说明 |
|------|------|--------|------|
| 图片压缩 | 🟢 可用 | P0 | Canvas API 实现 |
| 格式转换 | 🟢 可用 | P0 | JPEG/PNG/WebP/HEIC |
| 尺寸调整 | 🟢 可用 | P1 | 等比/指定尺寸 |
| 添加水印 | 🟢 可用 | P1 | 文字水印 |
| 截图美化 | 🟢 可用 | P2 | 设备外壳、背景 |
| 图片拼接 | 🟢 可用 | P2 | 横向/纵向/网格 |
| 批量工作流 | 🟢 可用 | P2 | 自定义处理流程 |

### 辅助功能

| 功能 | 状态 | 优先级 | 说明 |
|------|------|--------|------|
| 国际化 (i18n) | 🟢 可用 | P1 | English / 中文，语言设置持久化 |
| 设置面板 | 🟢 可用 | P1 | 语言切换、版本信息 |
| 帮助文档 | 🟢 可用 | P1 | 快速开始、快捷键、FAQ |
| 剪贴板集成 | 🟢 可用 | P0 | 检测并导入图片 |

## 开发规范

### 代码规范
- Vue 3 Composition API
- 组件命名: PascalCase
- 样式: scoped CSS + CSS Variables
- 图标: @vicons/carbon

### Git 规范
- 分支: `main` + 功能分支
- Commit: `feat:`, `fix:`, `docs:`, `refactor:` 前缀
- 每次发布打 tag
