# ImageBox 开发计划

## 项目概述

- **项目名称**: ImageBox (cb-imagebox)
- **当前版本**: 0.0.1
- **目标**: 一款轻量级图片处理工具箱，支持压缩、格式转换、水印、裁剪、截图美化等功能

---

## 功能规划

### 核心功能

| 功能 | 状态 | 优先级 | 说明 |
|------|------|--------|------|
| 图片压缩 | ✅ 已完成 | P0 | Canvas API 实现，支持质量调节 |
| 格式转换 | ✅ 已完成 | P0 | JPEG/PNG/WebP 输出 |
| 尺寸调整 | 🔴 待开发 | P1 | 等比/指定尺寸 |
| 添加水印 | 🔴 待开发 | P1 | 文字/图片水印 |
| 截图美化 | 🟡 实现中 | P2 | 设备外壳、背景 |
| 图片拼接 | 🔴 待开发 | P2 | 横向/纵向/网格 |
| 批量工作流 | 🔴 待开发 | P2 | 自定义处理流程 |

### 辅助功能

| 功能 | 状态 | 优先级 | 说明 |
|------|------|--------|------|
| 国际化 (i18n) | 🔴 待开发 | P1 | English / 中文 |
| 设置面板 | 🔴 待开发 | P1 | 语言切换、版本信息 |
| 帮助文档 | 🔴 待开发 | P1 | Zoom、快捷键、FAQ |
| 剪贴板集成 | ✅ 可用 | P0 | 检测并导入图片 |

## 已完成功能详情

### 图片压缩
- 支持 JPEG/PNG/WebP 输入
- 输出格式可选 JPEG/PNG/WebP
- 质量调节 1-100
- 显示原文件大小、压缩后大小、节省比例
- 支持多图批量压缩

### 格式转换
- JPEG → PNG/WebP
- PNG → JPEG/WebP
- WebP → JPEG/PNG
- HEIC → JPEG/PNG/WebP (通过 heic-decode)
