# ImageBox 交互设计文档

## 全局交互

### 侧边栏导航

```
行为:
- 点击导航项切换到对应功能模块
- 当前项高亮显示 (primary 色背景)
- 鼠标悬停显示 tooltip

动画:
- 切换时主内容区淡入淡出 (200ms)
- 侧边栏图标悬停缩放 1.1x
```

### 剪贴板检测

```
触发: 应用获得焦点时
行为:
- 检测剪贴板是否有图片
- 有图片时底部弹出提示栏
- 点击"快速导入"自动粘贴到当前功能

动画:
- 从底部滑入 (300ms ease-out)
- 3秒无操作自动收起
```

### 文件拖拽

```
触发: 文件拖入窗口
行为:
- 拖拽经过时上传区域高亮
- 显示虚线边框 + 阴影
- 释放后验证文件类型
- 不支持的文件显示错误提示

视觉反馈:
- 拖拽中: 边框变为 primary 色
- 有效文件: 显示绿色勾选
- 无效文件: 显示红色警告
```

## 功能模块交互

### 1. 图片压缩

```
文件添加:
1. 用户选择文件 / 拖拽 / 粘贴
2. 显示文件列表（缩略图 + 信息）
3. 自动计算预估大小

质量调节:
- 拖动滑块实时更新预估大小
- 滑块旁显示当前百分比

开始压缩:
1. 按钮变为 loading 状态
2. 显示进度条（批量时）
3. 完成后显示结果对比
4. 提供"打开目录"和"继续添加"选项

结果展示:
- 原始 vs 压缩后并排显示
- 显示节省空间百分比
- 点击下载或复制到剪贴板
```

### 2. 添加水印

```
实时预览:
- 任何设置修改立即更新预览
- 预览区域可缩放查看细节

位置选择:
- 点击九宫格位置立即生效
- 支持自定义偏移量输入

文字水印:
- 输入框实时更新预览
- 支持回车换行
- 颜色选择器使用原生

图片水印:
- 上传水印图片
- 可调节大小和透明度
- 支持平铺模式

批量应用:
- 保持相同设置应用到所有图片
- 每张图片单独预览切换
```

### 3. 格式转换

```
格式选择:
- 点击卡片选择目标格式
- 选中卡片高亮边框
- 根据格式显示相关选项

选项联动:
- 选 PNG/WebP 时显示透明通道选项
- 选 JPEG 时显示质量滑块
- 选 GIF 时显示动画帧率选项

HEIC 处理:
- 检测到 HEIC 自动显示提示
- 转换为兼容格式后处理
```

### 4. 尺寸调整

```
模式切换:
- 指定尺寸: 输入宽/高（保持比例联动）
- 百分比: 统一缩放比例

尺寸联动:
- 勾选"保持宽高比"时
- 修改宽度自动计算高度
- 修改高度自动计算宽度

预设选择:
- 点击预设自动填充数值
- 支持添加自定义预设

预览:
- 显示原始尺寸虚线框
- 显示新尺寸实线框
- 标注缩放比例
```

### 5. 截图美化

```
设备选择:
- 点击设备卡片切换外壳
- 预览区实时渲染效果

样式调节:
- 滑块调节内边距/圆角/阴影
- 实时预览效果

背景:
- 渐变: 选择预设渐变或自定义
- 纯色: 使用颜色选择器
- 透明: 导出 PNG 带透明

导出:
- 支持导出为 PNG/JPG
- 可批量导出多张截图
```

### 6. 批量工作流

```
流程编辑:
- 点击"+ 添加步骤"弹出选择器
- 拖拽步骤调整顺序
- 点击步骤展开编辑面板

预设使用:
- 点击预设自动填充流程
- 可基于预设二次编辑

批量处理:
- 支持文件夹导入
- 保持目录结构输出
- 实时显示处理进度
- 支持暂停/继续/取消

命名规则:
- 支持变量: {原文件名}, {日期}, {序号}
- 实时预览命名示例
```

## 状态与反馈

### 加载状态

```
按钮 loading:
- 显示 spinner 图标
- 文字变为"处理中..."
- 禁用点击

区域 loading:
- 显示骨架屏
- 或显示半透明遮罩 + spinner

全屏 loading (批量处理):
- 显示进度条
- 显示当前处理文件名
- 显示预计剩余时间
```

### 成功反馈

```
Toast 通知:
- 位置: 右上角
- 持续时间: 3秒
- 内容: 操作成功 + 简要信息

完成动画:
- 勾选动画
- 数字增长动画（节省空间）

自动操作:
- 处理完成后可选自动打开目录
- 或自动复制到剪贴板
```

### 错误处理

```
文件错误:
- 不支持的格式: 红色提示 + 跳过
- 文件损坏: 标记错误 + 继续其他
- 权限问题: 提示用户授权

网络错误:
- 显示重试按钮
- 保留当前进度

输入验证:
- 实时验证输入
- 无效时输入框变红
- 显示具体错误信息
```

## 快捷键

```
全局:
- Ctrl/Cmd + V: 粘贴剪贴板图片
- Ctrl/Cmd + O: 打开文件选择
- Ctrl/Cmd + A: 全选列表
- Delete: 删除选中项
- Esc: 关闭弹窗/取消操作

功能内:
- Space: 预览大图
- Ctrl/Cmd + S: 保存结果
- Ctrl/Cmd + Z: 撤销（编辑时）
```

## 空状态

```
无文件时:
- 显示大号图标
- 提示文字 + 操作按钮
- 示例: "拖拽图片到这里开始"

无历史记录:
- 显示空状态插图
- 提示: "还没有处理记录"

搜索结果空:
- 提示: "没有找到匹配的文件"
- 提供"清除筛选"按钮
```

## 过渡动画

```
页面切换: 200ms ease-out fade
列表项添加: 300ms ease-out slide-up
列表项删除: 200ms ease-in slide-left
模态框: 300ms cubic-bezier(0.4, 0, 0.2, 1)
Toast: 300ms ease-out slide-right
进度条: 100ms linear
```
