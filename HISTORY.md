# Version History

## v0.0.3 (2025-03)

- Complete all core features:
  - Image compression with quality control and batch support
  - Watermark support (text watermark with position/opacity/font customization)
  - Format conversion between JPG, PNG, WebP, GIF, HEIC and more
  - Image resize with aspect ratio control and presets
  - Screenshot beautify with device frames, gradients and shadows
  - Image join in horizontal, vertical and grid layouts
  - Batch workflow with custom processing pipelines
- Settings panel with language and theme options
- Help panel with quick start guide and keyboard shortcuts

## v0.0.2 (2025-02)

- Internationalization (vue-i18n): English (en) and Simplified Chinese (zh-CN)
- Settings panel: language switching, theme toggle (light/dark), version info
- Help panel: version info, quick start guide, zoom usage, keyboard shortcuts, supported formats, tips
- Settings persistence via localStorage

## v0.0.1 (2024-03-27)

- Initial release
- Project setup with Vue 3 + Vite
- Naive UI component library integration
- Basic UI layout with top nav and sidebar
- Image compression via Canvas API (quality control, batch, JPEG/PNG/WebP)
- Format conversion support
- HEIC format support via heic-decode
- Clipboard image paste
- Zoom support
